# Rock Paper Scissors Game in Python

A simple command-line Rock Paper Scissors game built using Python. The user plays against the computer, and the computer makes random choices.

---

##  Features

-  Rock
-  Paper
-  Scissors
-  Computer chooses randomly
-  Win, Lose, or Draw result
-  Beginner-friendly Python project

---

## Technologies Used

- Python 3
- `random` module
- Dictionaries
- Conditional Statements

---

## How to Run

1. Install Python 3
2. Clone the repository
